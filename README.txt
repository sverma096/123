
LexVoice Studio - SaaS Style Voice Legal Drafting App

Run:
1. pip install -r requirements.txt
2. python app.py
3. Open http://127.0.0.1:5000

Features:
- Modern SaaS UI
- Voice typing (Hindi / English / Hinglish)
- Live preview using backend formatting
- Export to Word (.docx) or PDF
- Legal/A4 paper support
- Signature block and stamp placeholder
